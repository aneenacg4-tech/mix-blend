This project is a simple web page showcasing a full-screen hero section with a background image and overlaid text using **Tailwind CSS**. The design is responsive and blends the background image with a dark overlay for better text readability.

## Features

- Full-screen background image with overlay effect.
- Responsive hero section text for different screen sizes.
- Tailwind CSS for styling.
